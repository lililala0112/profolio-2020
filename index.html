About me

[ 資深的設計能力 ]
在學生時期學習於設計相關知識，畢業後從事網頁設計師與UI設計師職位長達七年時間，經歷過上百件的形象網站與知名電商平台，國家公園政府標案UI設計，以及能力挑起公司主設計的大樑與帶領新生設計師，專案經理與工程師一同討論完成專案。

[ 前端程式能力 ]
為了更了解網頁這領域的知識讓跨部門溝通更順暢，督促我學習程式這領域，在工作之餘參加程式相關的課程(Jquery , PHP)。在既有的 HTML / CSS 能力與 JS 程式的初步學習，在一個契機之下，回台中開始前端工程師的職務，接觸的是與過往設計師不同的面貌，一直不斷積極參與多項與前端相關課程補足自己前端程式的技能樹，也積極參與 台中前端社群 並加入組織幹部，三年的實質的前端工程師經驗，快速迭代的Scrum工作模式，與版控協同合作，後端工程師配合串接資料，介面規劃互動效果功能平台，前端程式工作與後端工程師配合的經驗讓我充分掌握整體的網頁技術。

[ 整合規劃與應用能力 ]
因為自己職涯的選擇不想只做程式鑽研，整合的工作類型會比較符合自身的規劃與興趣所在，現階段是自由工作者，自己能力接案雖然小型，但透過獨立接案的過程從第一線去了解客戶的需求，透過不同的案源利用競業分析初步了解其產業與規劃架構，建立功能架構與估價，掌握進度，時程規劃，使用者體驗研究，頁面設計，前端串接資料，Git版控，與後端工程師配合完成專案。 使用者脈絡與需求質性分析與設計思考發想，將痛點轉換成機會點的功能，運用Sketch轉化成有形的圖像Wireframe，易用性測試Prototype反覆測試使用者需求。
